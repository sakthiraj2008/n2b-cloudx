# N2B CLOUD

A file & video hosting platform using:
- Supabase (Auth + DB)
- Telegram (Storage - upcoming)
- Razorpay (Payments)

## Features
- Login / Signup
- Upload files
- Video streaming
- Daily upload limit
- Pro plan
- Admin dashboard

## Setup
1. Add Supabase URL & Key
2. Create tables (users, files)
3. Run SQL functions
4. Open index.html
